## Installatie
<!-- Bij Installatie staat hoe een andere developer aan jouw repo kan werken -->

```
 "dependencies": {
    "dotenv": "^16.4.7",
    "express": "^5.1.0",
    "liquidjs": "^10.21.0"
  }
```

1. Fork en Clone de repo
2. In Terminal: NPM install express
3. In Terminal: NPM install liquidjs
4. In Terminal: NPM install dotenv
